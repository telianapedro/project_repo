# project_repo# Dashboard de Análise de Veículos

## Descrição
Aplicativo web interativo para análise de dados de anúncios de vendas de carros, desenvolvido com Streamlit durante o Sprint 5 do curso de Análise de Dados.

## Funcionalidades
- 📊 Histogramas interativos
- 📈 Gráficos de dispersão
- 🎛️ Controles interativos (botões e seleções)
- 📱 Interface responsiva

## Acesso ao Aplicativo
🚀 [Ver aplicativo online]((https://project-repo-e8tj.onrender.com/))

## Tecnologias Utilizadas
- Python
- Streamlit
- Pandas
- Plotly Express

## Como executar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```
